h1. A log watcher written in Python

Allows RegEx-based line matching for various actions.

Configs are JSON files of the following format:

bc. [
    {
        "regex": "<pattern>",
        "action": "<action>"[,
        "<action_option>": "<value>"[, ...]
        ]
    }[, ...]
]

Currently supported ConsoleUI actions:

* *Print* (with colours!)
** with options _fore_, _back_, and _style_ (see "colorama":https://pypi.python.org/pypi/colorama for a list of supported values)

* *Ignore*


h2. Example config for monitoring SASS output:

bc. [
    {
        "regex": "^>>>.*",
        "action": "Print",
        "fore": "GRAY"
    },
    {
        "regex": "^\\s+error.*",
        "action": "Print",
        "fore": "WHITE",
        "back": "RED"
    },
    {
        "regex": ".*",
        "action": "Print",
        "fore": "YELLOW",
        "style": "DIM"
    }
]
